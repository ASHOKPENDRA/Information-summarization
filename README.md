# Information Summarization

Information Summarization is a web-based application that generates concise
summaries from text and documents using an external summarization API.  
The project is developed using PHP and JavaScript with a MySQL database
connection and follows secure coding practices.

---

## Features
- Text and document summarization
- API-based summarization integration
- PHP-based backend logic
- MySQL database connectivity
- Multiple PHP pages (~15+) for application workflow
- Vendor-based summarization tools
- Secure handling of sensitive credentials
- Responsive web interface using HTML & CSS (via PHP)

---

## Technologies Used
- PHP
- JavaScript
- HTML5
- CSS3
- MySQL
- Cohere API
- Vendor libraries for summarization

## Setup Instructions
1. Clone this repository
2. Create config.php using config.sample.php
3. Add MySQL database credentials
4. Add your summarization API key
5. Import the database (if provided)
6. Run the project on a PHP server (XAMPP / InfinityFree)

---

## Security Practices
- API keys and database credentials are stored in config.php
- Sensitive files are excluded from GitHub using .gitignore
- Sample configuration file is provided for reference
